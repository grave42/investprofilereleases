# InvestProfile — Releases

Официальный репозиторий релизов **InvestProfile** — офлайн-приложения для отслеживания инвестиционного портфеля на Московской бирже (MOEX).

> Исходный код: [grave42/InvestProfile](https://github.com/grave42/InvestProfile)

---

## Что такое InvestProfile

InvestProfile — десктопное приложение для Windows, которое заменяет таблицы Google Sheets при управлении портфелем ценных бумаг. Работает полностью офлайн (цены подтягиваются с MOEX ISS API без авторизации), все данные хранятся локально в SQLite.

**Возможности:**
- Группировка активов по разделам: Акции, Фонды, Облигации, Валюта
- Автообновление цен с Московской биржи по расписанию
- Расчёт плановых лотов, доли, дельты и P&L по каждой позиции
- Круговая диаграмма аллокации по секторам
- Логотипы эмитентов
- Экспорт / импорт данных портфеля
- Поддержка валютных курсов (СNY, USD, EUR)
- Тёмная тема, единственный режим

---

## Установка

1. Перейдите в раздел **[Releases](../../releases)** этого репозитория.
2. Скачайте файл `.exe` (установщик) из последнего релиза.
3. Запустите установщик — приложение установится и появится ярлык на рабочем столе.
4. При первом запуске укажите общую стоимость портфеля в настройках.

> **Требования:** Windows 10 / 11 (64-bit). Интернет нужен только для обновления цен с MOEX.

---

## Обновление

Скачайте установщик из нового релиза и запустите его поверх старой версии. База данных и настройки сохраняются.
