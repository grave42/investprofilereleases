# InvestProfile — Releases

Официальный репозиторий релизов **InvestProfile** — офлайн-приложения для отслеживания инвестиционного портфеля на Московской бирже (MOEX).

---

## Что такое InvestProfile

InvestProfile — десктопное приложение для Windows, macOS и Linux, которое заменяет таблицы Google Sheets при управлении портфелем ценных бумаг. Работает полностью офлайн (цены подтягиваются с MOEX ISS API без авторизации), все данные хранятся локально в SQLite.

**Возможности:**
- Группировка активов по разделам: Акции, Фонды, Облигации, Валюта
- Автообновление цен с Московской биржи по расписанию
- Расчёт плановых лотов, доли, дельты и P&L по каждой позиции
- Круговая диаграмма аллокации по секторам
- Логотипы эмитентов
- Экспорт / импорт данных портфеля
- Поддержка валютных курсов (СNY, USD, EUR)
- Тёмная тема, единственный режим

## Установка

Перейдите в раздел **[Releases](../../releases)** и скачайте файл для своей платформы.

### Windows
1. Скачайте `InvestProfile-Setup-x.x.x.exe`
2. Запустите установщик — появится ярлык на рабочем столе
3. При первом запуске укажите общую стоимость портфеля в настройках

> **Требования:** Windows 10 / 11 (64-bit)

### macOS
1. Скачайте `InvestProfile-x.x.x.dmg`
2. Откройте `.dmg` и перетащите приложение в папку **Applications**
3. При первом запуске: **правой кнопкой → Открыть** (обход Gatekeeper)

> **Требования:** macOS 10.15 Catalina и новее (64-bit)

### Linux
1. Скачайте `InvestProfile-x.x.x.AppImage`
2. Сделайте файл исполняемым и запустите:
```bash
chmod +x InvestProfile-*.AppImage
./InvestProfile-*.AppImage
```

> **Требования:** Ubuntu 20.04+ или совместимый дистрибутив (64-bit)

> Интернет нужен только для обновления цен с MOEX.
